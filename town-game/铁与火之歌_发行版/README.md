# 铁与火之歌 — 19世纪欧洲手工艺人传奇

一款以19世纪欧洲工业革命为背景的文字冒险游戏。你将扮演一位16岁的孤儿，被老匠人格雷收为学徒，在炉火与铁砧之间，用锤子、剪刀、镊子和活字，锻造出属于自己的命运。

## 背景设定

**时间**: 1842年，伦敦东区

工业革命的巨变正在席卷欧洲。机器大生产以摧枯朽之势冲击着延续数百年的手工业体系。曾经受人尊敬的匠人们，如今面临着前所未有的生存危机。

工厂的烟囱遮蔽了天空，机器的轰鸣声取代了锤击的韵律。行会制度早已瓦解，手工艺人们失去了庇护，只能各自为战。

然而，正是在这最黑暗的时刻，一场名为"工艺美术运动"的反抗悄然兴起——不是用枪炮，而是用匠心。

## 游戏特色

### 七阶晋升体系

```
学徒 → 帮工 → 匠人 → 师傅 → 大师 → 宗师 → 巨匠
```

从最基础的学徒开始，通过完成订单、积累声望、学习技能，逐步成长为独当一面的匠人大师。

### 四大职业路线

| 职业 | 简介 | 技能分支 |
|------|------|----------|
| **铁匠** | 在炉火与铁砧之间锻造力量 | 武器锻造 / 农具制作 / 装饰铁艺 |
| **裁缝** | 一针一线缝合布料与命运 | 贵族礼服 / 民间服饰 / 戏剧服装 |
| **钟表匠** | 在齿轮的滴答声中窥见时间 | 怀表修理 / 精密仪器 / 自动人偶 |
| **印刷工** | 活字之间承载知识与革命 | 书籍装帧 / 海报设计 / 秘密出版 |

### 具体化的技能系统

每个技能都对应一件可见的实物和一段生动的故事：

- **铁匠**的"第一柄短剑"：三昼夜的反复锤打，你终于将一块铁料变成了一柄短剑
- **裁缝**的"丝绸之手"：当丝绸在阳光下流动如水，罗丝夫人倒吸一口气
- **钟表匠**的"修复怀表"：当秒针重新跳动，老人泪流满面
- **印刷工**的"革命之声"：当印刷品在工人中流传，你已站在历史的一边

### 丰富的NPC系统

- **格雷师傅** — 你的师父，满手老茧，目光如炬
- **艾丽丝** — 同门学徒，竞争对手也是朋友
- **霍尔先生** — 商人公会代表，圆滑而精明
- **罗丝夫人** — 贵族遗孀，对手工艺品有极高鉴赏力
- **艾奇** — 地下记者，揭露工厂的黑暗

### 经济与声望系统

- **金币** — 完成订单获得，用于购买材料和工具
- **声望** — 影响NPC态度和可解锁的剧情
- **经验** — 积累后可升级技能和属性

## 运行方式

### 环境要求

- Python 3.8+
- 无需安装任何第三方库

### 启动游戏

```bash
cd town-game
python main.py
```

## 操作说明

| 操作 | 说明 |
|------|------|
| 输入数字选择 | 在菜单中输入对应数字进行选择 |
| 查看技能树 | 查看已习得和可学习的技能 |
| 接单工作 | 完成订单获得金币和经验 |
| 逛市场买材料 | 购买工具和材料提升属性 |
| 与NPC对话 | 推进剧情，获得任务和信息 |
| 保存并休息 | 保存游戏进度 |
